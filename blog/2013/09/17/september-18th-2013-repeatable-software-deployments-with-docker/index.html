September 18th, 2013: Repeatable Software Deployments With Docker

Sep 17th, 2013

The September meet up is on Wednesday, September 18th, hosted at Reynolds Journalism Institute. The talks start at 6:00. We meet in the glass room right in the main lobby.

Repeatable Software Deployments with Docker

James Carr

Imagine being able to take your application, all of its system level depenencies and being able to package it all up and give it to someone and say “Here, run this.” Linux Containers help provide this kind of capability but are very difficult to use and understand. Docker makes it very easy to build, ship and run containers to allow for 100% repeatable deployments.

James Carr is a DevOps Engineer at Zapier and spends his time automating infrastructure provisioning with puppet and occasionally hacks on product development in Django and node.js.

This month’s food is sponsored by Zapier.
