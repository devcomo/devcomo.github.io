July 17th, 2013 - Application Monitoring and Objective C

Jul 11th, 2013

The July meet up is on Wednesday, July 17th, hosted at Reynolds Journalism Institute. The talks start at 6:00. We meet in the glass room right in the main lobby.

Using New Relic For Server Monitoring

James Carr

Setting up an extensive monitoring solution for your application can be time consuming and difficult. Tonight I’ll demonstrate using New Relic to keep tabs on server performance, monitor and debug production applications and show how to use the recently launched platform to monitor all of the services in your stack. I’ll also give a brief demonstration of how to record deployments to track application performance changes that result from new code roll outs.
James Carr is a DevOps Engineer at Zapier and spends his time automating infrastructure provisioning with puppet and occasionally hacks on product development in Django and node.js.

Modern Objective C

Bryan Pratte and Brett Koonce

We will look at some modern Objective-C techniques (ARC, Blocks) and some traditional methods that many people do not use (Categories, Extensions). Also, we will take a peek at iOS 7 and Xcode 5.
