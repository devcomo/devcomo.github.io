August 2012: Getting Started With Django and Lightning Talks

Aug 9th, 2012

August’s meetup is on Wednesday, August 15th! Here’s the plan: it will be hosted at our usual Museao location (directions). Talks start at 6:00pm but come by around 5:30pm or so if you want to hang out beforehand.

Come join us on August 15th.  Based on our member survey, a lot of people are interested in learning more about Python and Django.  Greg Aker is going to stop by and share some tricks in a ‘Getting Started with Django’ session.

Getting started with Django

By: Greg Aker

Django is Python’s “Swiss Army Knife” web development framework. Let’s look at how easy it is to get up and running, be productive and have fun developing complex web applications!

About Greg

By day, Greg Aker (@gaker) works as Chief Code Thrasher at nGen Works building awesome things in Django. He also slings the terminal as a systems administrator at EngineHosting. 

 

Lightning Talks

By: ???

Come by and do a short presentation on anything you are interested on.  Have a new tool or technology that you have been using and want to show off?  Know some cool text editor tricks that you want to share?  Take the stage and present it.  We assume lightning talks will be <= 5 minutes.

Hope to see everyone at Museao this month!
